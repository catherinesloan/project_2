About

The Australian Marriage Law Postal Survey was an optional survey run between 12th September and 17th November 2017. The survey asked “should the law be changed to allow same-sex couples to marry?”, with respondents required to mark either the ‘Yes’ box or ‘No’ box.

This webpage explores the results of the survey in relation to the 2016 Federal Election, electorate characteristics and survey participation.

The home page displays three visualisations that can be filtered by State:

  • The Pie chart represents the overall percentage of ‘Yes’ and ‘No’ votes in the Marriage Postal Survey, per state.
  • The horizontal stacked bar chart visualises the survey respondents by age. Each bar represents a different electorate from the selected state.
  • The bubble chart explores the Marriage Postal Survey, the Federal Election and electorate characteristics. Each bubble represents an electorate from the selected state, where the size of the bubble is determined by the percentage of ‘Yes’ votes for that electorate in the Marriage Postal Survey. The y-axis value represents the percentage of the population within that electorate with a higher education. The percentage of labor votes in the Federal Election for each electorate is represented on the x-axis.  This percentage is out of the total Labor and Liberal votes for the electorate, so represents a scale of whether each electorate is more Labor leaning or Liberal leaning. A colour scale is also applied so that the colour of the bubbles changes from blue to red depending on how Labor or Liberal that electorate voted.


The ‘Map’ option in the navigation bar leads to a choropleth map which shows the Federal Electorates coloured in proportion to the percentage of ‘Yes’ responses in the Marriage Postal Survey. The map was created using the choropleth leaflet plugin and a Geojson file. It also features a leaflet loading symbol plugin. This tells the user when the page is loading after a zoom button has been selected, or the map has been moved.
  • The Leaflet-choropleth plugin can be found here.
  • The Leaflet.loading plugin can be found here.


The ‘Data’ tab in the navigation bar has a drop down menu with an option for five data sources. The first three options link to the tables used to create the visualisations on the ‘Home’ page. The fourth data source is a table of characteristics and statistical information that was used for the map, while the final data source is the GeoJSON.
